## 可选配置

- `conf.cssModuleTransform.styleVariable` 
  - 类型: `string` 
  - 默认: "Style"
- `conf.cssModuleTransform.ignorePrefix` 
  - 类型: `string` | `string[]` 
  - 默认为空
